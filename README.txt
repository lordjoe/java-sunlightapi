Sunlight Labs Java Api

Sunlight.jar is an executable jar with a series of test calls. It  includes the
sources of the library from json.org http://www.json.org/java/ you
can run it by saying:
> java -jar Sunlight.jar 

JavaApiDocs.ZIP is javadoc and can be unzipped and hosted on the web site

SunlightSrc.jar has sources only including the  json.org sources

SunlightOnly.jar has sources and classes but drops the json.org sources
